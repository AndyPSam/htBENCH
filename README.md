# htBENCH
A Multithreaded Web Server Benchmarking Tool

## Usage

```
cd src
make all
./htbench -h <hostname> -p <port_no> -d <duration> -t <no_of_threads> -n <no_of_request>
```
> `sh htbench.sh`
